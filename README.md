# お問い合わせユーザーの管理システム
ユーザーに自身の情報とご意見を入力してもらい、それをデータベースに登録する。
登録してあるデータをもとに検索システム(絞り込み)を使用しデータを表示する。

## 作成した目的
COACHTECHのテスト

## 機能一覧
- 名前で検索
- 性別で検索
- 登録日で検索
- メールアドレスで検索

## 使用技術(実行環境)
- Laravel 8.83.19
- jQuery
